# **App Name**: AdivinaExamen

## Core Features:

- Subida de Documentos: Permite a los usuarios subir documentos como exámenes anteriores, temarios y apuntes en diversos formatos (PDF, DOCX, etc.).
- Análisis de Contenido: Analiza el contenido de los documentos subidos para identificar temas recurrentes, preguntas frecuentes y la estructura general del curso. El análisis determinará la importancia relativa de los temas.
- Predicción de Preguntas: Utiliza algoritmos de machine learning para predecir posibles preguntas de examen basadas en el análisis de contenido. La predicción se basa en la frecuencia de los temas, su importancia y patrones en exámenes anteriores.
- Explicación IA: Implementa un modelo de lenguaje que, dada una pregunta predicha, genera una respuesta detallada y explicativa, simulando la explicación de un profesor. El modelo actuará as a tool and decide if some parts of the explanation should be incorporated to the response, depending on the type of question and topic.
- Interfaz de Estudio: Interfaz intuitiva para visualizar las preguntas predichas y sus respuestas generadas por la IA, con opciones para estudiar y marcar preguntas como aprendidas.

## Style Guidelines:

- Color primario: Azul suave (#64B5F6), transmite confianza y seriedad.
- Color de fondo: Gris claro (#F0F4F8), proporciona un fondo neutro que no distrae.
- Color de acento: Verde (#4CAF50), destaca elementos importantes y acciones.
- Tipografía clara y legible, adecuada para la lectura prolongada de texto educativo.
- Iconos intuitivos y representativos para las diferentes funciones de la aplicación.
- Diseño limpio y estructurado, que facilita la navegación y el acceso a la información.
